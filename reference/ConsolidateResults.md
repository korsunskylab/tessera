# Consolidate Tessera results from multiple samples (groups) after constructing Tessera tiles separately on cells from each group.

Consolidate Tessera results from multiple samples (groups) after
constructing Tessera tiles separately on cells from each group.

## Usage

``` r
ConsolidateResults(res, group.by)
```

## Arguments

- res:

  Output of running GetTiles (when consolidate == FALSE).

- group.by:

  Name of metadata variable that identifies distinct groups.
